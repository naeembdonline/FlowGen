🔍 TROUBLESHOOTING:
--------------------------------------------------------------------------------

Problem: "SSH connection refused"
Solution: Check server IP, ensure SSH is running
Command: ssh root@72.60.234.138 "echo 'test'"

Problem: "Permission denied"
Solution: Verify SSH password/keys
Command: ssh root@72.60.234.138

Problem: "Port already in use"
Solution: Check what's using ports 3000/3001
Command: netstat -tulpn | grep :3000

Problem: "SSL certificate fails"
Solution: Ensure DNS points to server
Command: nslookup leads.naeemmia.bd

Problem: "Container won't start"
Solution: Check logs and disk space
Command: docker logs flowgen-backend
Command: df -h
